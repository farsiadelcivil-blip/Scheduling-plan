۰. تنظیمات اتصال به گیت‌هاب (فقط یک‌بار لازم است)

چطور توکن بسازم؟

گیت‌هاب → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → دسترسی Repository permissions → Contents: Read and write → فقط روی همین ریپو محدودش کن.

۵. زمان‌بندی

با کلیک روی این دکمه، پروژه به فایل projects.json در ریپو اضافه می‌شود. گیت‌هاب اکشن به‌صورت خودکار اجرا می‌شود: رویدادهای کلندر ساخته می‌شوند و ایمیل برنامهٔ زمان‌بندی مستقیماً برای منتور(ها) ارسال می‌شود (چند دقیقه طول می‌کشد).
